项目信息

运行环境：

开发工具：IDEA 2019.3.2

JDK版本：11.0.4

数据库：Mysql/Redis(缓存)

前端框架：Layui

前端模板：KITADMIN

模板地址：http://www.jq22.com/demo/BeginnerAdmin201701192304/#/

后端框架：Spring Boot

安全框架：Shiro(集成Redis缓存、MD5加密、Kaptchar验证码、thymeleaf标签)
